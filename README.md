# 🚗 ESP32 Bluetooth Controlled RC Car

## 📌 Description

This project is a Bluetooth-controlled RC car built using ESP32 and Dabble GamePad module. It allows real-time wireless control of a robot using a smartphone.

## ⚙️ Features

* Forward / Backward / Left / Right movement
* PWM speed control using ESP32
* Bluetooth communication via Dabble App

## 🔧 Technologies Used

* Embedded C (Arduino)
* ESP32 PWM (ledc)
* Bluetooth Communication

## 📱 How to Use

1. Upload code to ESP32
2. Install Dabble App
3. Connect to "MyBluetoothCar"
4. Use GamePad to control

## 🔌 Pin Configuration

* Right Motor: 16, 17, 22
* Left Motor: 18, 19, 23

## 🚀 Future Scope

* Obstacle Avoidance
* ESP32-CAM Integration
* Voice Control
